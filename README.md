# Электронная система учёта успеваемости

Дипломный проект по специальности 09.02.07 «Информационные системы и программирование».

## О проекте

Веб-приложение для автоматизации учёта оценок и посещаемости студентов. Система предоставляет доступ для четырёх ролей: администратор, преподаватель, студент и родитель.

## Технологии

- **Backend:** Python, FastAPI, SQLAlchemy, SQLite
- **Frontend:** HTML5, CSS3, JavaScript (без фреймворков)
- **Аутентификация:** JWT-токены, bcrypt
- **Графики:** Chart.js
- **Экспорт:** OpenPyXL (Excel)

## Функциональность

- Аутентификация и авторизация (4 роли)
- Управление группами, студентами, преподавателями, предметами
- Выставление оценок (экзамен, зачёт, курсовая, практика)
- Учёт посещаемости с визуализацией
- Формирование ведомостей группы
- Экспорт данных в Excel
- Логирование действий с фильтрацией
- Тёмная тема оформления
